Supabase Database Setup Instructions
===================================

1. Log in to your Supabase account at https://supabase.com

2. If you haven't created a project yet:
   - Click "New Project"
   - Enter a name for your project
   - Set a database password (save this somewhere secure)
   - Select a region closest to you
   - Click "Create Project"

3. Wait for your project to be created (this may take a few minutes)

4. Once your project is ready, go to the SQL Editor:
   - In the left sidebar, click on "SQL Editor"

5. Run the database schema:
   - Copy the contents of the file "supabase/ready_to_run_schema.sql"
   - Paste it into the SQL editor
   - Click the "Run" button

6. After the schema is created, you can verify the tables were created:
   - In the left sidebar, click on "Table Editor"
   - You should see the following tables:
     * users
     * products
     * transactions
     * transaction_items
     * activity_logs

7. Update your environment variables:
   - In the left sidebar, click on "Project Settings" (gear icon)
   - Click on "API"
   - Copy the "Project URL" and "anon public" key
   - Update your .env file with these values

8. Test your setup:
   - Start your Next.js application: npm run dev
   - Visit http://localhost:3000
   - Try logging in as admin (admin@example.com / Password123)
   - Try logging in as cashier (cashier@example.com / Password123)
